
AChecker module provides reporting on the WCAG compliance of node content.


Installation
------------

Copy achecker.module to your module directory and then enable on the admin
modules page. You need to set AChecker session ID which you can get at http://achecker.ca to use the module.  
Module configuration at admin/config/achecker.

Configuration
-------------

Also at the module configuration page you can set the compliance standard and fields to validate.
